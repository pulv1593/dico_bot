# dico_bot
youtube 기반 음원을 사용하는 노래봇입니다.
여러개의 서버에서 개별적으로 사용은 가능하나 사용량이 많아질수록 성능 문제로 배포 예정X 
(애초에 개인적으로 지인 서버에서만 사용할 용도로 제작되었음.)

구현 기능
1. 음악 재생(/p)
   /p youtube url
   -url의 음원을 재생.
   
   /p 검색할 키워드
   -youtube에 검색후 최상단 3개의 음원 중 택1 하여 해당 음원 재생.
   검색 후 0번 입력시 검색 취소 가능.
   
2. 노래 스킵(/skip)
   현재 재생중인 노래 skip하기
    
3. 대기열 확인(/queue)
   현재 대기열에 담긴 노래 목록 확인.

필요시 추가적인 기능 구현 예정.
